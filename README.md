# Portfolio Web - Beatriz Christine

Portfolio pessoal desenvolvido para apresentar minha trajetoria, habilidades, projetos, certificados e formas de contato.

## Preview

O projeto é um site feito com HTML, CSS e JavaScript.

```text
https://BeatrizChristine.github.io/PortifolioWeb/
```

## Tecnologias

- HTML5
- CSS3
- JavaScript
- Design responsivo

## Funcionalidades

- Seção inicial com apresentação
- Sobre mim
- Carrossel de projetos
- Habilidades tecnicas
- Certificados
- Links de contato
- Curriculo em PDF

## Estrutura do Projeto

```text
PortifolioWeb/
|-- assets/
|   |-- docs/
|   |-- icons/
|   `-- img/
|-- css/
|   |-- animation.css
|   |-- responsive.css
|   `-- style.css
|-- js/
|   `-- script.js
|-- index.html
|-- .gitignore
`-- README.md
```

## Como Rodar Localmente

1. Baixe ou clone este repositorio.
2. Abra o arquivo `index.html` no navegador.

Também é possível usar a extensão Live Server no VS Code para visualizar o site com recarregamento automático.
