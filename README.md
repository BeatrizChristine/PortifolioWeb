## Como Visualizar Localmente

Para abrir o projeto no computador:

1. Clone este repositorio:

```bash
git clone https://github.com/BeatrizChristine/PortifolioWeb.git
```

2. Entre na pasta do projeto:

```bash
cd PortifolioWeb
```

3. Abra o arquivo `index.html` no navegador.

Tambem e possivel usar a extensao Live Server no VS Code para visualizar o site com recarregamento automatico.

## Publicacao

Este projeto pode ser publicado com GitHub Pages.

Depois de enviar os arquivos para o GitHub:

1. Acesse o repositorio no GitHub.
2. Va em `Settings`.
3. Entre em `Pages`.
4. Em `Build and deployment`, selecione `Deploy from a branch`.
5. Escolha a branch `main` e a pasta `/root`.
6. Salve as alteracoes.

O GitHub Pages ira gerar um link publico para acessar o portfolio.
